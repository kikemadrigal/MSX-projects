# test input system assembler code

Let's test the input system of the MSX1

To test the project you will have to create a directory called tools that contains the openmsx and sjasm folders (see the make.bat file)

Finally write make in the Windows CMD

Para probar el proyecto tendrás que crear un directorio llamado tools que contenda las carpetas de openmsx y sjasm (mira el archivo make.bat)

Por último escribir make en el CMD de Windows
